# Diet-Note 飲食管家 - 關注您的健康
這是一個可以記錄及觀察每日飲食內容的應用程式。
<br/>輸入每日餐點的組成及熱量，您的飲食習慣。
<br/>適合每一個想要紀錄飲食內容和觀察飲食組成的每一個人。
